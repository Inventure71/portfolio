ReminderZ

(Development name: RemainderV0) View on GitHub

ReminderZ (formerly RemainderV0) is a desktop application designed to help users manage messages, notes, reminders, and projects, with AI-powered features for organization and content generation. It utilizes a Python backend with a pywebview frontend (HTML, CSS, JavaScript) for a cross-platform user experience.

Core Features

  • Message Management: Efficiently store, organize, and retrieve messages, notes, and creative ideas. Assign items to user-defined projects and attach relevant images.
  • Project Organization: Create, customize (name, description, color, emoji), and manage projects. View messages filtered by project for focused work.
  • AI Integration (Gemini Model):
    • Engage in contextual AI chat.
    • Automate message processing: assign to projects, identify potential reminders.
    • Receive suggestions for new projects based on unassigned content.
    • Generate textual descriptions for uploaded images.
    • Select messages relevant to user queries via AI.
  • Reminders: Set precise reminders with specific dates, times, and recurrence options (daily, weekly). Receive desktop notifications (macOS specific) for timely alerts.
  • Clipboard Integration (macOS specific): Monitor the system clipboard and automatically save copied text to a dedicated "Saved Clips" project after a configurable number of copies.
  • Telegram Integration: Fetch messages and attachments directly from a configured Telegram bot, storing them in the application's database and as JSON logs.
  • WhatsApp Scraper Integration: Trigger a utility script to scrape messages from WhatsApp Web (current implementation prints to console).
  • Image Handling: Upload and associate images with messages. View AI-generated descriptions for visual content.

Key Technologies

  • Backend: Python
  • Frontend: HTML, CSS, JavaScript
  • GUI Framework: pywebview (embedding web content in a native GUI window)
  • AI Model: Google Gemini
  • Database: SQLite (for local data storage)
  • macOS specific features: AppKit (via PyObjC for clipboard and status bar), osascript (for desktop notifications)
  • Web Scraping: Selenium (for WhatsApp automation)

Project Structure Highlights

The application is modularly structured:

  • main.py: Core backend logic and pywebview initialization.
  • web/: Contains all frontend assets (HTML, CSS, JS), including pages and reusable components.
  • DatabaseUtils/: Python modules for SQLite database interactions (messages, projects, clipboard).
  • Utils/: Houses utility modules like the AI model handler (model_handler.py), clipboard monitor, reminder scheduler, and integration scripts (Telegram, WhatsApp).
  • API_keys/: Stores API credentials (e.g., gemini_api_key.json).
  • settings.json: Application settings.

Setup & Running

To get ReminderZ running locally:

  1. Ensure Python 3.x is installed.
  2. Obtain a Gemini API key and place it in API_keys/gemini_api_key.json.
  3. For Telegram features, configure BOT_TOKEN in a .env file.
  4. Clone the repository and install dependencies: 
    pip install -r requirements.txt
    (Note: The original README mentioned a requirements.txt file needs to be generated. Likely dependencies include pywebview, google-generativeai, requests, python-dotenv, AppKit, selenium, python-dateutil.)
  5. Run the application: 
    python main.py

Note: macOS specific features require running on macOS. WhatsApp scraping requires Google Chrome.

Building

The application can be packaged into a standalone executable using PyInstaller with the provided RemainderApp.spec file:

pyinstaller RemainderApp.spec
